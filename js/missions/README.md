# Limit 180 — 50關卡題目生成概要與擴充指南 (js/missions/)

本目錄存放了《Limit 180》所有 50 個關卡 (Mission 1-50) 的隨機題目生成與註冊邏輯。

為提升可維護性並避免單一檔案過大，所有關卡生成演算法皆已按特訓階段拆分為四個獨立的小元件檔案：**[phase1.js](phase1.js)**、**[phase2.js](phase2.js)**、**[phase3.js](phase3.js)** 與 **[phase4.js](phase4.js)**。這些演算法會由核心題目生成器 `js/generator.js` 動態路由調度。

## 📈 50 個關卡出題規劃與特色

本特訓系統將關卡規劃為 **四個階段**，實現從奠基到極致超頻的難度進程。
每個關卡接續 20 個 Stage（關卡階梯）。其中：
* **Stage 1 ~ 5**：**無時間限制**。不進行倒數，提供無壓力的數字直覺反射練習。
* **Stage 6 ~ 10 / 11 ~ 15 / 16 ~ 20**：分別對應逐漸緊縮的**單題倒數秒數**限制（具體配置見下方表格）。

## 🚀 擴充新關卡 (以新增 Mission 51 為例)

擴充新關卡非常直覺，只需要 **三個步驟**：

### 步驟一：於對應的階段檔案中實作生成函數
打開對應的階段腳本（例如第四階段 [js/missions/phase4.js](file:///Users/oshukezu/Documents/Knowledge%20Vault/Game/limit-180/js/missions/phase4.js)，或新建立一個 `phase5.js` ），在閉包函數內最下方掛載新關卡：

```javascript
  // Mission 51: 新關卡說明
  Gen.missions[51] = function() {
    const a = randInt(10, 99);
    const b = randInt(1, 9);
    const qText = `${a} + ${b} = ?`;
    const ans = (a + b).toString();
    
    return {
      type: 'calc',                    // 題型：'calc' (計算) 或 'compare' (比大小)
      questionText: qText,
      correctAnswer: ans,
      explanation: `${a} + ${b} = ${ans}。`,
      key: `M51:add:${a}:${b}`         // 必須唯一的防重 key
    };
  };
```

### 步驟二：於全域配置中註冊關卡設定
打開 [js/game-config.js](file:///Users/oshukezu/Documents/Knowledge%20Vault/Game/limit-180/js/game-config.js)，在 `MISSION_CONFIGS` 中加入關卡參數與 Stage 區間秒數：

```javascript
  const MISSION_CONFIGS = {
    // ... 前 50 關 ...
    50: { name: "Mission 50", totalQuestions: 3, desc: "東大特訓班 100 題完全體", stages: { 1: 9999, 2: 5, 3: 2.5, 4: 9 } },
    
    // 新增第 51 關
    51: { 
      name: "Mission 51", 
      totalQuestions: 3,                                // 測試題數
      desc: "全新特訓主題描述", 
      stages: { 
        1: 9999,                                        // Stage 1~5 倒數 (>=999 代表自由計時)
        2: 8,                                           // Stage 6~10 倒數 (秒/題)
        3: 5,                                           // Stage 11~15 倒數 (秒/題)
        4: 3                                            // Stage 16~20 倒數 (秒/題)
      } 
    }
  };
```

### 步驟三：在 HTML 中引入腳本
若您是直接在現有的 `phase4.js` 中新增，可跳過此步。若是建立了全新的 `phase5.js`，請在 [index.html](file:///Users/oshukezu/Documents/Knowledge%20Vault/Game/limit-180/index.html) 中引入它：
```html
  <script src="js/missions/phase5.js"></script>
```

儲存後重新整理頁面，大廳網格便會自動渲染出新關卡卡片並可正常運行遊玩！
